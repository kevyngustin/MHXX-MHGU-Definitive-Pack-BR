# MHXX-MHGU-Definitive-Pack-BR
Pacote completo para emulação de Monster Hunter Double Cross (3DS) e Generations Ultimate (Switch). Inclui tradução PT-BR para MHXX, updates, cheats/mods de performance e um conteúdo denso com todas as DLCs que encontrei para ambas as versões, incluindo missões de evento, Palicos especiais e bônus do MH4U.
